# AI ENGINEERING KNOWLEDGE BASE: GENIUS LEVEL (V14)

## 1. Термостатические системы (Thermostatic Technology)
- **Восковой элемент (Wax Element):** Стандарт для GROHE TurboStat. Реагирует на перепады давления за 0.3–1.0 сек. Допуск температуры ±1°C.
- **Безопасность:** SafeStop (блокировка на 38°C), CoolTouch (изоляция горячих каналов внутри латунного корпуса, чтобы поверхность смесителя оставалась холодной).

## 2. Финишные покрытия (PVD - Physical Vapor Deposition)
- **Процесс:** Осаждение паров титана/циркония в вакуумной камере при 500°C.
- **Характеристики:** Твердость до 2500 HV по Виккерсу (в 3-10 раз тверже гальванического хрома). Устойчивость к царапинам и кислотным чистящим средствам.
- **Цвета PVD:** Hard Graphite, Brushed Bronze, Cool Sunrise (Золото), Polished Nickel.

## 3. Архитектура скрытого монтажа (Concealed Installation)
- **GROHE Rapido SmartBox (35600000):** 
  - Глубина монтажа: 75–105 мм.
  - Подключения: 3 выхода сверху (1/2"), 2 входа снизу (1/2").
  - Юстировка панели: до 6° по горизонтали.
- **Hansgrohe iBox Universal (01800180):**
  - Глубина монтажа: 80–108 мм.
  - Подключения: 4 порта Rp 3/4" (DN20).
  - Особенности: Ротационная симметрия (любая сторона - верх). Звукоизолирующий корпус.

## 4. Инсталляции для подвесного санфаянса (Concealed Cisterns)
- **Geberit Duofix:** Самонесущая стальная рама. Стандартная нагрузка = 400 кг.
- **Бачки Sigma:** 
  - Sigma 12 см (UP320) - стандарт.
  - Sigma 8 см (UP720) - для узких перегородок.
- **Совместимость:** К инсталляциям Sigma подходят ТОЛЬКО клавиши серии Sigma (Sigma01, Sigma20, Sigma50 и т.д.). Клавиши Delta НЕ подходят.

## 5. Санфаянс и Гигиена (Ceramics & Hygiene)
- **Rimless / Безободковый смыв:** GROHE Triple Vortex (тройной водоворот), Roca Rimless.
- **KeraTect (Geberit):** Специальная глазурь без микропор, предотвращающая налипание бактерий.
